# AyumeeAiGamesBot

Bot Telegram AI gaya **oWo game** dengan inventori aesthetic, hunt, battle, shop, dan chat AI.

## Fitur
- /start → menyapa pemain
- /train → simulasi training AI
- /inventory → menampilkan inventori aesthetic
- /hunt → hunting loot random
- /battle → melawan musuh random
- /shop → daftar item di shop
- Chat biasa → AI membalas gaya oWo
